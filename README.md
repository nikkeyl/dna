# DNA Simulator: Образовательный проект

## О проекте

Привет! Это образовательный симулятор для тех, кто хочет разобраться в основах ДНК и молекулярной биологии, но не готов читать скучные и тяжелые учебники.

Наша главная цель — сделать науку простой, наглядной и увлекательной. Приложение показывает, как устроена ДНК и как работают базовые механизмы человеческого генома.

⚠️ **Важное уточнение:** Проект **не генерирует реальные последовательности ДНК** и не опирается на настоящие медицинские базы данных. Это исключительно песочница для обучения и экспериментов.

## Задачи проекта

- **Доступная наука:** Объясняем сложные генетические процессы простым человеческим языком.
- **Наглядность:** Интерактивно показываем структуру ДНК (аденин, тимин, гуанин, цитозин) и то, как они связаны друг с другом.
- **Безопасная среда:** Вы можете смело изучать геном, не переживая о сложных правилах работы с реальными биоданными.

## Технологии

Приложение построено на базовых веб-технологиях:

- HTML5
- CSS3
- Vanilla JavaScript
